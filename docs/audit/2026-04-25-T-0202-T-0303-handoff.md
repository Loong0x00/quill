### 类型隔离原则 (T-0302/0303 共建, **接班核心约束**)

**alacritty 类型彻底锁在 `src/term/mod.rs` 内部**:
- `CellPos { col: usize, line: usize }` ↔ alacritty Point, 私有 `fn from_alacritty` (saturating cast `.max(0) as usize`)
- `CursorShape` enum 5 variants ↔ alacritty CursorShape, 私有 `fn from_alacritty` (exhaustive match 无 `_`)
- 任何下游 `pub use alacritty_terminal::*;` 都是 P0 违规, grep 应零命中
- 私有 inherent fn 而非 `impl From` trait — 否则下游 use alacritty 类型就能反向构造, 类型隔离失效
- exhaustive match 无 `_ =>` — 编译期 catches alacritty 加新 variant 的回归

T-0305 加 fg/bg 时 Color 必须 quill 自定义 struct (不 re-export alacritty Color), 沿袭同套路。
